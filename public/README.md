## ğŸš€ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/mgs007/quick-e-commerce-store.git
cd quick-ecommerce-store
```

### 2. Install Dependencies

```bash
# Using npm
npm install

# Or using yarn
yarn install

# Or using pnpm
pnpm install

# Or using bun
bun install
```

### 3. Start the Development Server

```bash
# Using npm
npm run dev

# Or yarn
yarn dev

# Or pnpm
pnpm dev

# Or bun
bun dev
```

### 4. Build for Production

```bash
npm run build
```

### 5. Preview Production Build

```bash
npm run preview
```
